# 说明
Centos7系统安装。

### 初始命令

#### 查看ip
    ip addr

#### 更新yum源
    yum -y install

#### 安装初始工具包
    yum -y install net-tools

#### 安装上传下载
    yum -y install lrzsz

#### 安装wget
    yum -y install wget

#### 安装vim
    yum -y install vim*

#### 安装压缩包
    yum install -y bzip2

#### 安装gcc依赖
    yum -y install gcc
    yum -y install gcc-c++

### 安装svn

    yum -y install subversion
